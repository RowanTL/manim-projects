## How to build

Install [uv](https://docs.astral.sh/uv/getting-started/installation/).
I run linux so mileage may vary for different systems.

1) `uv venv`
2) `source .venv/bin/activate`
3) `uv sync`
4) `make`

The video should appear in the `media` folder.
