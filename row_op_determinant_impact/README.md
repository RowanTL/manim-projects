## How to build

Follow the installation instructions for [ManimCE](https://docs.manim.community/en/stable/installation/uv.html).

1) `uv venv`
2) `source .venv/bin/activate`
3) `uv sync`
4) `make render`

The video will appear in the `media` folder.
