# Overview

There is some terminology to classify the solution space of a linear system of equations. Is the system *consistent* or
*inconsistent*? If the system is *consistent*, is the solution *unique*? This short illustrates this how these solutions
come to be.

An extremely simple video for an extremely simple concept.

[https://www.youtube.com/shorts/5EQBJOpfZHM](https://www.youtube.com/shorts/5EQBJOpfZHM)

## How to build

Follow the official instructions to install [ManimCE](https://docs.manim.community/en/stable/installation/uv.html). The
user should follow the instructions to install [uv](https://docs.astral.sh/uv/getting-started/installation/).

1) `uv venv`
2) `source .venv/bin/activate` if on linux. Windows has a different command.
3) `uv sync`
4) `manim render EntireShort`
